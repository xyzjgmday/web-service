# Project: Lumen Client App

# 📁 Tugas 15

## Project setup

### Clone Tugas 15

```sh
git clone -b tugas-15 https://github.com/xyzjgmday/web-service.git
```

```
composer install
```

### Database: Migrations

```
php artisan migrate
```

### Database: Seeding

```
php artisan db:seed
```

### Local Development Server

```
php -S localhost:5000 -t public
```
