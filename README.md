# Project: Lumen Service App
Tugas 5 - Web Service

## Installation
### Git Clone
```sh
git clone -b tugas-5 https://github.com/xyzjgmday/web-service.git
```

### install Dependency 
```sh
composer install
```
### Migrate 
```sh
php artisan migrate
```

### Sending Seeder's
```sh
php artisan db:seed
```

### how to running
```sh
#port 5000
php -S localhost:5000 -t public
```
or
```sh
#port 500
php artisan serve
```

## End-point: Getting Status
### Method: GET
>```
>localhost:500
>```

## End-point: Login
### Method: POST
>```
>localhost:500/login
>```
### Body (**raw**)

```json
{
    "email" : "daysubhiz@gmail.com",
    "password" : "password"
}
```
