# Project: Lumen Client App

# 📁 Tugas 12

## Project setup

### Clone Tugas 13

```sh
git clone -b tugas-13 https://github.com/xyzjgmday/web-service.git
```

```
composer install
```

### Database: Migrations

```
php artisan migrate
```

### Database: Seeding

```
php artisan db:seed
```

### Local Development Server

```
php -S localhost:5000 -t public
```

## End-point: Get Posts

### Method: POST

> ```
> localhost:5000/public/posts
> ```
